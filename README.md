# To-Do List Advanced

An advanced, interactive To-Do List web application built with HTML, CSS, and JavaScript.  
This project includes features like categories, filters, and persistent storage using LocalStorage.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Categorize tasks
- Filter tasks by category or status
- Responsive design for mobile and desktop
- Lightweight and fast
